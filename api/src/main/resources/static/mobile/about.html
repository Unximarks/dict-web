臧彩楼，男，小学文化。1967年4月20日出生于内蒙古兴安盟科右前旗大坝沟乡。现住呼伦贝尔市。中共乌尔其汉镇个私联合支部书记，牙克石市乌尔其汉镇华泰饮品公司董事长、养老院院长、华夏尚品商贸有限公司总经理、全国志愿助残阳光使者、诚信企业家、优秀共产党员。几年来，曾先后被中国民营企业管理委员会授予“优秀民营企业家”称号，先后获得呼伦贝尔市优秀共产党员、“创先争优先进工作者”、“残疾人自强模范”、“兴国百名英才”等诸多荣誉称号。获第五次全国自强模范称号，并受到了习近平主席的接见。

他是一名二级肢体残疾人，曾经有过坎坷的人生经历。他没有向命运低头，靠着党的好政策和政府的扶持，走出一条艰苦创业之路，现在经营一家有着50多名员工的华泰饮品有限公司，创办了一家福利养老院。

生活中难以想象的灾祸，不止一次降临到臧彩楼的身上：与妻子冲破重重阻碍最终走到一起，婚后却被火车轧去左腿；生活刚好转，一场突如其来的大火使他仅有的右脚重度烧伤；随后弟弟的离去，母亲精神失常让他备受打击；女儿的降生本以为是幸福的开始，厄运又一次降临，大年初二妻子被歹徒砍死在家中，女儿被砍成重伤，最终离去。

臧彩楼虽然命运坎坷，但他生活在充满了爱的社会里。为救女儿走投无路的时候，当创业陷入危机的时候，是党和热心的哈尔滨市民一次又一次的无私帮助使他渡过难关，这坚定了他回报社会的信念。

从2000年起，他经常利用节假日，背上修理箱，到一家敬老院免费给老人们修理钟表眼镜。

2013年，他创办一所孤残人员托养公寓，已经安置16名残疾人、孤寡老人，都是免费入住托养。

他经常购买一些木料、水泥等，义务帮助困难家庭维修房屋。

经常买米买面，送到生活困难的残疾人家中。

此外，他还为一些看不起病的孤残人士提供医疗费用。

对于社会的无私回报，激起了社会强烈反响，给臧彩楼带来了各种荣誉。
