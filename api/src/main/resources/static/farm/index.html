公司概述

      建乐生态农场位于湖南省湘潭县白石镇深溪村，创立于2016年。主营水稻、红薯、南瓜、湘莲、藠头、油菜、辣椒等农作物。农场经营主要采用公司+散户的合作模式，目前耕种总面积高达4000亩，是湘潭县境内较大的绿色生态农作物种植基地。生产方式运用了间套作、复种、轮作等方式，让过去的蔬菜种植由一年1季变为一年2季、3季甚至多季，改革了传统种植模式，也提升了单位面积产量和效益。

市场需求

      2018年长沙黄兴海吉星交易量达639万吨，交易额达588亿元， 日均交易量稳定在12000吨以上，稳坐全国蔬菜批发市场“头把交 椅”同时，海吉星承担了长沙市90%以上的蔬菜供应任务，保障了 全省80%的蔬菜供给，是湖南名副其实的“菜篮子”。藠头是人们喜闻乐见的可口美味的时令蔬菜，可鲜食炒肉，也可做酸菜泡菜藠头， 营养价值丰富。光海吉星市场，日销藠头高达上万斤，因此市场需求量极大。同理，红薯，辣椒，南瓜，紫苏，藕尖等时令蔬菜的需求容量同样惊人。 近几年，国家大力支持三农建设，特别是对于农业生产的农户，给予政策税收等方面的优惠，增加了农户的生产积极性。 随着生活水平的提高，食品安全被人们更加重视。特别是绿色环保蔬菜瓜果的需求量与日俱增。

种植生产过程

大量开垦荒田，减低生产成本。种植过程中，运用了间套作、复种、轮作等方式，让过去的蔬菜种植由一年1季变为一年2季、3季甚至多季，改革了传统种植模式，也提升了单位面积产量和效益。 以水稻和紫苏套种种植为例，上半年垦地种植水稻，一季收成之后，再加种紫苏，待成熟，即可收成。 基于环保考虑，农场多采用轮作方式，比如今年种旱季作物，明年改种水季作物，这种方式便于休田，保持土壤肥效，作物更环保，口感更优越，产量更高。
