asked questions

Answer Particular Topic

我们是什么？

建乐生态农场采用现代化种植经营模式，生产绿色无公害环保农作物，主要作物有水稻、紫苏、南瓜、红薯、藠头、油菜等。 农场经营采用公司+散户的合作模式，生产方式运用了间套作、复种、轮作等方式，目前耕种总面积高达4000亩，是湘潭县境内较大的绿色生态农作物种植基地。

我们种什么？

水稻、紫苏、南瓜、红薯、藠头、油菜

我们怎么做？

1.运用“套种”技术、
2.机械化耕作
3.农业生产技术输出

我们已经做了什么？

2017年种植湘莲30亩 创造营收约3万元 种植50亩油菜，产油约1500斤 创造营收约1.5万元。
2018年种植50亩油菜，产油约1500斤 创造营收约1.5万元 种植水稻油菜、红薯 创收约5万元。
2019年 增产种植藠头50亩 创造营收约5万元 种植水稻50亩，产出大约60000斤稻子，创造营收约7万元。
