农业发展环境

    随着人民的生活水平不断提高，对健康生态的绿色环保食材越加关注，生态有机环保蔬菜粮食的市场需求越来越大。建乐生态农场积极推广绿色生态农业模式，为消费者提供真正无公害、新鲜、营养、优质、安全、无农药残留的农产品。
今年是一个特殊年份，一场新冠肺炎的爆发让人们更加意识到粮食和农业生产的重要性。习近平总书记讲，越是面对风险挑战，越要稳住农业，越要确保粮食和重要副食品安全。因此，农业生产在政策方面，是极度利好的。

机会与挑战

机会：“三农”问题是全党工作的重中之重，国家政策将大力扶持农业生产,周边乡镇，大规模种植的农户较少，竞争对手比较薄弱，目前农田承包成本低。
挑战: 农业靠天吃饭的现实问题难以改观,人力成本高、农业技术薄弱等……

奋斗目标

1.在原有种植基础上，扩大再生产。审时度势调整种植结构。 2.除了自产，采用合作社的形式，带动当地农户生产。 3.为广大农户提供技术服务。 4.根据湖南红壤特性，种植药材，如白术，金银花等。 5.尝试直播带货。

asked questions

Answer Particular Topic

我们是什么？

建乐生态农场采用现代化种植经营模式，生产绿色无公害环保农作物，主要作物有水稻、紫苏、南瓜、红薯、藠头、油菜等。 农场经营采用公司+散户的合作模式，生产方式运用了间套作、复种、轮作等方式，目前耕种总面积高达4000亩，是湘潭县境内较大的绿色生态农作物种植基地。

我们种什么？

水稻、紫苏、南瓜、红薯、藠头、油菜

我们怎么做？

1.运用“套种”技术、
2.机械化耕作
3.农业生产技术输出

我们已经做了什么？

2017年种植湘莲30亩 创造营收约3万元 种植50亩油菜，产油约1500斤 创造营收约1.5万元。
2018年种植50亩油菜，产油约1500斤 创造营收约1.5万元 种植水稻油菜、红薯 创收约5万元。
2019年 增产种植藠头50亩 创造营收约5万元 种植水稻50亩，产出大约60000斤稻子，创造营收约7万元。
